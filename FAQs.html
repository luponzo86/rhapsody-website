RAPSODY

Re-Assessment of Pathogenicity of SAVs based On Dynamics

  1. How can I obtain the PolyPhen-2 output file and use it on RAPSODY?

    • Go to PolyPhen-2 batch query page
    • Submit your list of variants and click on "Submit Batch"
    • In the next screen, click the "Refresh" button until your job appears in the "Batches" field as completed
    • Click on "Full" link under "Results" column and save the content in a text file (example).
      NB: the "Logs" link contains useful information about possible errors encountered during the analysis of variants.
    • Upload the saved file to RAPSODY (option 1)
    • Optional: provide your email to get notified when the job is completed
    • Click on "Submit job"
    • Wait until job status changes to completed


  2. How can I interpret the results?

    3. Once the analysis is completed, the results page will show a link where to visualize the predictions for each variant that was successfully processed, and a link to download all results (in both Unix and Windows format). A description for each file is provided in the README file.


  3. What if I don't have a PolyPhen-2 output file? What does "predictions based only on structural/dynamical features" (option 2) mean?

    4. The random forest classifier normally used for predictions is trained on both sequence-based conservation features and structural/dynamical features. We extract the conservation features from the output of PolyPhen-2, while the structural/dynamical ones are computed internally.
    If a valid PolyPhen-2 output is not provided, we can still run a reduced version of the classifier that employes only structural/dynamical features. The accuracy of predictions is however lower than that achievable by the full classifier, so we recommend this approach only if one is not interested in accurate predictions but just wants to compute dynamical features.
    Moreover, contrary to PolyPhen-2, we can only process variants in Uniprot format (e.g. "P17516 135 G E" or "SUN2_HUMAN 671 G R").
    The "batch query" option can be used to submit multiple queries, written in a text file.
